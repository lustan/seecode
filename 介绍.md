# Seecode：浏览器里的小型 IDE，专治各种"代码碎片"

写代码的人，电脑里多多少少都囤着一堆零散东西：

- 接口返回的 JSON，想格式化看看结构；
- 同事甩来两段配置，要对比一下哪儿不一样；
- 临时想记几行 SQL、Python 片段，又懒得开 VS Code。

每次为这点事打开一个重量级编辑器，说实话挺费劲。所以我做了 **Seecode**——一个常驻在浏览器里的代码小工具，开一下就能用，关掉也不占地方。

> 一句话：**JSON 格式化 + 多语言便签 + 文件对比**，三件套全在 Chrome 插件里搞定。

## 🔥 4.0 最想吹的：文件对比

这次更新最爽的就是这个，灵感直接抄的 JetBrains IDEA 那个 Compare Files。

- **左右两栏都能编辑**，不是只能看的死面板，边改边对比，diff 实时刷新。
- **LCS 算法**算行级差异，红色删除、绿色新增、黄色修改，一眼分清楚。
- **滚动联动**，左边滚到哪儿，右边自动锚到对应的位置，不用手动对齐。
- **F7 跳下一个差异，Shift+F7 跳上一个**，键盘党狂喜。
- 顶上还有个 `−/~/+` 统计条，删了几行、改了几行、加了几行清清楚楚。
- 一键 **Swap** 翻转左右，**Format JSON / Copy / Clear** 每栏独立按钮。

平时调接口对比两版返回值、Code Review 之前先自己看看 diff，特别顺手。

## ✨ 其他几个常用功能

### 📝 便签管理

- 支持 **JSON、JavaScript、Java、Python、SQL、Markdown**，写啥都不别扭。
- **自动识别语言**，你粘进去什么它就高亮成什么。
- 侧边栏可以 **拖拽排序**，常用的拽到顶上。

### 🛠️ JSON 工具

- 一键 **Format**，遇到带尾逗号、单引号那种"半坏"的 JSON 也能修。
- **Compact 模式**一键压成单行，准备贴回代码里很方便。

### 🪟 悬浮便签

把任何一条笔记 **拖出来变成独立小窗口**，永远置顶。一边看接口文档一边写代码，再也不用切来切去。

### 🔎 搜索替换

`Ctrl+F` 搜索，`Ctrl+R` 替换，支持 **正则** 和 **区分大小写**，跟 IDE 一个手感。

### ⚡ 性能 & 体验

- 基于 **CodeMirror 6**，几千行的 JSON 丢进去也不卡。
- 主题切换：**IDEA Dark** / **GitHub Light**，跟着你心情走。

## ⌨️ 常用快捷键

| 快捷键 | 作用 |
| --- | --- |
| `Ctrl + J` | 格式化 |
| `Ctrl + K` | 压缩 JSON |
| `Ctrl + F` / `Ctrl + R` | 搜索 / 替换 |
| `Ctrl + D` / `Ctrl + Y` | 复制 / 删除当前行 |
| `F7` / `Shift + F7` | 下一个 / 上一个差异 |
| `Esc` | 关闭面板 |

## 🚀 怎么装

目前是开发者模式手动加载，几步就完事：

1. 把仓库 clone 或者下载下来。
2. 打开 Chrome，地址栏敲 `chrome://extensions/`。
3. 右上角打开 **开发者模式**。
4. 点 **加载已解压的扩展程序**，选项目根目录。

OK，浏览器工具栏就多了个图标，点开就能用。
